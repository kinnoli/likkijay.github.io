Emmet

General information

Abbreviations syntax

Create element

p // Outputs <p></p>
em // Outputs <em></em>

Child elements (>)

div>ul>li // Outputs <div> <ul> <li></li> </ul> </div>

Sibling elements (+)

div+p // Outputs <div></div> <p></p>

Climb-up (^)

div>p>em^p // Outputs <div> <p> <em></em> </p> <p></p> </div>

Multiplication (*)

ul>li*3 // Outputs <ul> <li></li> <li></li> <li></li> </ul>

Grouping

div>(header>h1+p)+footer>p*2 // Outputs <div> <header> <h1></h1> <p></p> </header> <footer> <p></p> <p></p> </footer> </div>

ID and CLASS

div#header // Outputs <div id="header"></div>
div.class1.class2.class3 // Outputs <div class="class1 class2 class3"></div> td[title="Hello" colspan="3"] // Outputs <td title="Hello" colspan="3"></td>
